用户接入

用户接入是指，用户可以直接进入您的网站或应用，无需注册和登录流程。

接入流程

  1. 参考左侧 开发者流程 创建应用。
  2. 参考 Oauth 2.0 协议获取AccessToken
  3. 通过AccessToken获取用户基本信息
  4. 关联用户基本信息到您的网站

获取用户信息

请参考上面的接入流程获取AccessToken，本章将介绍通过AccessToken获取用户基本信息。

  • 网址:https://auth_host/apis/oauth2/v1/userinfo
  • 方法：GET
  • 请求参数: Authorization: Bearer {access_token}
  • 请求示例（cURL）：

如果您获取到的 AccessToken 为 a14afef0f66fcffce3e0fcd2e34f6ff4，那请求为：

   curl "https://auth_host/apis/oauth2/v1/userinfo" -H "Authorization: Bearer a14afef0f66fcffce3e0fcd2e34f6ff4"
  • 返回结果：（HTTP状态码为：200）
 {
   "uid": "4f6ff4af66fcffce3e0fcd2e3",
   "name": "小明"
 }

这个包含的用户的基本信息，如果想获取更多个人信息，请参考 API 资源列表/获取用户数据 。

  • 错误处理： （HTTP状态码为：非200）
    {
      "error": "bad_request",
      "error_description": "error_description"
    }
