开发者使用流程

本文档为开发者接入索引，如果您是第一次使用开发者中心，可以参考该文档快速上手开发者中心。

创建应用

为了保护用户的数据，防止API被滥用或恶意使用，豆瓣要求每个API的使用者申请一个应用，而每个应用唯一标识一个API使用者。你可以在申请页面创建一个新应用，填写必要信息后提交，就能得到你的应用。

申请数据权限

本着为了保护用户的数据的目的，所有的用户数据权限申请必须经过用户同意，高级权限需要管理员审核。

在您的应用内，诱导用户进去【获取 authorization code 的链接 】下面的网址，即可向用户发起权限申请，示例网址中的scope参数为userinfo，用于获取用户基本信息，申请界面如下。

用户点击同意后，您就获得了对应权限，用户再次进入时，则不会弹出权限申请窗口。如果您的应用，在运行过程中需要其他权限，可以添加scope参数，再次向用户发起申请。

注意事项

为了防止用户数据滥用，请根据实际情况申请用户权限，一般情况下用户基础信息就可以满足大部分使用场景，我们的工作人员会定期对用户授权进行审查，如果您申请了与使用场景非必要的用户权限，您的应用将会被停用或删除。
