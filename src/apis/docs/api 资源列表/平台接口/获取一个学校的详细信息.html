获取一个学校的详细信息

接口地址：/v1/schools/{sid}

返回格式：JSON

请求方式：GET

请求头部：

     Content-Type : application/json
     Authorization: {Bearer Token}

请求参数：

无

返回结果：

{
    \"schoolid\" : \"235\",
    \"region\": \"华北地区\",
    \"schoollogo\": \"http://xxxxxxxx.jpg\",
    \"schoolname\": \"中国人民大学\",
    \"ownurl\": \"v.ruc.edu.cn\",
    \"schoolurl\": \"ruc\"   
}
返回字段值字段类型说明
schoolid整形学校ID
region字符串所处地区
schoollogo字符串学校logo
schoolname字符串学校名称
ownurl字符串学校域名
schoolurl字符串学校代号
