获取当前学校的部门列表

接口地址：/v1/schools/{sid}/departments

返回格式：JSON

请求方式：GET

请求头部：

     Content-Type : application/json
     Authorization: {Bearer Token}

请求参数：
请求字段值字段类型说明
perpage整形返回数据条数，默认为10
curpage整形返回当前页码数，默认为1
order字符串排序字段，默认为‘ctime desc’

返回结果：

{
    \"count\": \"110\",
    \"data\": [
        {
            \"ctime\": \"2015-09-02\",
            \"departmentname\": \"国际关系学院\",
            \"departmentid\": \"4223\",
        },
        {
            \"ctime\": \"2015-09-02\",
            \"departmentname\": \"信息学院\",
            \"departmentid\": \"4233\",
        }
        ...
    ]
}
返回字段值字段类型说明
count整形部门总数
data数组当前页部门数据数组
data->ctime日期部门创建时间
data->departmentname字符串部门名称
data->departmentid整形部门ID
