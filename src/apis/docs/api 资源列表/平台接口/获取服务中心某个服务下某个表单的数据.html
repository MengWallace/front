返回字段值字段类型说明
aid整形服务ID
atime日期表单签批日期
autocheck整形0或1，服务是否自动受理
available整形0或1，服务当前是否可填写
begintime日期服务开始申请时间
categoryid整形服务分类ID
categoryname字符串服务分类名称
categoryimg字符串服务分类图标
costtime字符串表单受理所花费时间
checktime字符串服务获审批时间
count整形服务申请表单数
ctime日期服务创建时间
departmentid整形服务所属部门ID
departmentname字符串服务所属部门名称
endtime日期服务停止申请时间
fieldInfoObject服务自定义字段结构
filterconfObject服务过滤器结构
idenid整形填表人身份ID
hastodo整形0或1，是否有待办事项
innerFieldInfoObject服务系统字段结构
intro字符串服务简介
maxnum整形服务允许同时提交的表单数量，若为0表示不限制
mtime日期服务修改日期
ptime日期表单提交日期
schoolid整形服务所属学校
scope整形服务开发范围ID
score整形服务所获评分
sname字符串服务名称
status整形表单状态ID
statusname字符串表单状态名称
userObject服务创建者基本信息
