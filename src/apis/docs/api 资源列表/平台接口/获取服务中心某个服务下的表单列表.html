获取服务中心某个服务下的表单列表

接口地址：/v1/servtbls/{aid}/records

返回格式：JSON

请求方式：GET

请求头部：

     Content-Type : application/json
     Authorization: {Bearer Token}

请求参数：
请求字段值字段类型说明
perpage整形返回数据条数，默认为10
curpage整形返回当前页码数，默认为1
order字符串排序字段，默认为‘ctime desc’

返回结果：

返回字段值字段类型说明
count整形该服务表单总数
data数组表单数据数组
data->ctime字符串表单填写时间
data->departmentname整形表单填写人所在部门
data->deptid整形表单填写人所在部门ID
data->number字符串表单号
data->rid整形表单ID
data->status整形表单状态ID
data->statusname字符串表单状态明文
data->uid字符串表单填写人UID
data->uname字符串表单填写人姓名
