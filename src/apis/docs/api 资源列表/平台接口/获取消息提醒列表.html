获取消息提醒列表

  • 请求地址： /v1/sysmsgs

  • 请求方法： get

  • 请求参数：

参数名说明
uid消息的接收人uid
type消息的类型对应平台 默认为全部 1提醒 2待办 3全部
result消息的状态 默认为未处理 0所有 1未处理 2已处理
source消息的来源 默认为全部 ‘me’:me,‘serv’:服务中心,‘publichome’:公共主页,‘group’:群组,‘calender’:日历,‘edu’:课程中心,‘comments’:评论,‘appcenter’:应用中心
curpage当前页码
perpage每页条数
