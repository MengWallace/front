获取某个学校某个类别的通知列表

  • 接口地址：/v1/schools/{schoolid}/notices
  • 返回格式：JSON
  • 请求方式：GET
  • 请求头部： Content-Type : application/json Authorization: {Bearer Token}

  • 请求参数：
    参数名称参数类型描述
    perpageint返回数据条数，默认为10
    curpageint返回当前页码数，默认为1
    typeint返回列表的通知类型
    uidint返回此UID能访问到的所有通知

    countint通知总数目

    data->schoolidint学校的ID
    data->nidint通知ID
    data->typeint通知类型
    data->fromuidint发布人uid
    data->fromnamestring发布人姓名
    data->logostring发布人头像
    data->departmentidint发布部门ID
    data->departmentnamestring发布部门名称

    data->contentstring通知内容
    data->starttimestring通知的开始时间
